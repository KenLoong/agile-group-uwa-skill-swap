Understanding post status

Skill posts can move through three simple states: open, matched, and closed.

Open

An open post is still available. Other students can read the post and express interest if they want to start a skill-swap conversation.

Open

Matched

A matched post means the owner has found a likely skill-swap partner or has started following up with someone. The post may still be visible, but it should not be treated as fully available.

Matched

Closed

A closed post is no longer active. It should not invite new interest, but it may still appear in the owner's dashboard or history.

Closed

Who can change a post status?

Only the owner of a post should be able to update its status. This prevents other users from closing or changing someone else's listing.

Tip: If your post is still looking for a partner, keep it open. If you have found someone, mark it as matched. If the opportunity is finished, mark it as closed.
